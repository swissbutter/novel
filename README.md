# 🟦 사각 스튜디오 (Sagak Studio)

> **웹소설 작가와 스토리텔러를 위한 올인원 시각적 설정 정리 및 집필 도구** > 별도의 설치나 서버 없이, HTML 파일 하나로 언제 어디서나 나만의 세계관을 구축하세요.

<br/>

## 📖 프로젝트 소개

**사각 스튜디오(Sagak Studio)**는 복잡한 세계관 설정과 집필 과정을 하나의 공간에서 해결할 수 있도록 설계된 웹 애플리케이션입니다.

인물, 사건, 장소 등의 설정을 **마인드맵(노드) 방식**으로 시각화하여 관리하고, 해당 설정을 보며 바로 글을 쓸 수 있는 **전용 에디터**를 제공합니다. 이 모든 기능이 **단 하나의 HTML 파일**에 담겨 있어, 인터넷 연결이 없는 환경에서도 완벽하게 작동하며 USB 등에 담아 휴대하기 간편합니다.

## ✨ 주요 기능

### 1. 🎨 비주얼 보드 (Visual Board)
복잡한 설정을 한눈에 파악할 수 있는 강력한 시각화 도구입니다.
- **다양한 노드 지원:** 인물, 사건, 장소, 아이템, 세력, 복선, 타임라인, 설정, 대사, 갈등 등 12종 이상의 전용 노드
- **직관적인 관계 연결:** 드래그 앤 드롭으로 노드 간의 관계를 잇고 설명을 추가 (실선/점선 지원)
- **스마트 레이아웃:** `Dagre` 알고리즘을 이용한 노드 자동 정렬 및 그룹화 기능
- **상세 데이터 관리:** 각 노드 타입에 최적화된 입력 필드 제공 (예: 인물-MBTI/직업, 아이템-등급 등)

### 2. ✍️ 집필 에디터 (Writing Editor)
설정에 집중하며 글을 써내려갈 수 있는 쾌적한 환경을 제공합니다.
- **멘션 시스템:** 에디터에서 `@` 또는 `#` 입력 시, 보드에 등록된 인물/사건 정보를 즉시 호출하여 자동 완성
- **집중 모드 (Zen Mode):** UI를 최소화하고 글쓰기에만 몰입할 수 있는 전체 화면 모드
- **목표 관리:** 목표 글자 수 설정 및 실시간 진행률(Progress Bar) 시각화
- **문서 상태 관리:** 초고, 수정, 완료 상태 태깅

### 3. 🛠️ 작가 편의 도구
- **이름 생성기:** 판타지(서양), 무협, 한국 현대물 등 장르별/성별 랜덤 이름 생성 기능 내장
- **커스터마이징:** 눈이 편한 **다크 모드** 및 다양한 한글 폰트(마루부리, 리디바탕, 갈무리 등) 지원
- **반응형 워크스페이스:** 드래그 앤 드롭으로 프로젝트 순서 변경 및 사이드바 정리

### 4. 🔒 데이터 관리 및 보안
- **로컬 저장:** 모든 데이터는 브라우저의 `LocalStorage`에 자동 저장됩니다.
- **강력한 보안 백업:**
  - `.json`: 일반 백업 파일
  - `.vel`: **AES 암호화**가 적용된 보안 파일 (비밀번호 설정 가능)
- **Word 내보내기:** 작성한 설정 데이터와 원고를 `.docx` 파일로 변환하여 다운로드

## 🚀 사용 방법 (Getting Started)

이 프로젝트는 **Single File Application**입니다. 별도의 빌드나 설치 과정이 필요 없습니다.

1. 저장소의 `board.html` 파일을 다운로드합니다.
2. Chrome, Edge, Whale 등 최신 웹 브라우저에서 파일을 실행합니다.
3. 바로 집필을 시작하세요!

<br/>

## 🛠️ 기술 스택 (Tech Stack)

외부 서버 의존성 없이 브라우저 클라이언트 환경에서만 동작하도록 구성되었습니다.

- **Core:** HTML5, React 18 (CDN), Babel Standalone
- **Styling:** Tailwind CSS (CDN)
- **UI/Animation:** Framer Motion
- **Libraries:**
  - `dagre`: 그래프 레이아웃 알고리즘
  - `crypto-js`: 데이터 암호화/복호화
  - `docx` & `FileSaver`: 워드 문서 생성 및 저장
  - `jszip`: 데이터 압축
