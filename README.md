# 📝 사각 스튜디오 (Sagak Studio)

**웹소설 작가와 스토리텔러를 위한 올인원 설정 관리 & 집필 도구**

사각 스튜디오는 복잡한 설치 과정 없이 **단 하나의 HTML 파일**만으로 브라우저에서 즉시 실행되는 웹 애플리케이션입니다. 인물 관계도 시각화부터 원고 집필까지 하나의 워크스페이스에서 해결하세요.

<br/>

## ✨ 주요 기능 (Key Features)

### 1. 🎨 비주얼 아이디어 보드 (Idea Board)
복잡한 세계관과 설정을 무한한 캔버스 위에 시각적으로 정리합니다.
- **노드 시스템**: 인물(👤), 사건(🔥), 메모(💡) 노드를 생성하고 자유롭게 배치합니다.
- **관계 연결**: 드래그 앤 드롭으로 노드 간의 관계를 잇고 설명을 추가할 수 있습니다.
- **자동 레이아웃**: 복잡하게 얽힌 인물 관계도(가계도)나 타임라인을 버튼 하나로 깔끔하게 정렬합니다.
- **상세 설정**: 캐릭터의 이름, 나이, 직업, 성별, 역할(주인공/적대자 등)을 체계적으로 관리합니다.

### 2. ✍️ 몰입형 집필 에디터 (Writer's Editor)
설정을 보며 원고를 작성할 수 있는 작가 전용 에디터입니다.
- **목표 관리**: 목표 글자 수를 설정하고 달성률 그래프를 실시간으로 확인합니다.
- **멘션 시스템**: `@인물명` 또는 `#사건명`을 입력하여 설정해둔 정보를 빠르게 참조하고 링크합니다.
- **집중 모드 (Zen Mode)**: 주변 UI를 숨기고 오직 글쓰기에만 집중할 수 있는 환경을 제공합니다.

### 3. 🛠️ 유틸리티 & 편의성
- **🎲 랜덤 이름 생성기**: 판타지, 무협, 현대물 등 장르와 성별에 어울리는 이름을 자동 생성합니다.
- **🔒 데이터 보안**: 작업물은 **AES 암호화(.vel)**를 지원하여 비밀번호로 안전하게 보호됩니다.
- **📄 내보내기**: 작업한 인물 설정과 원고 내용을 `.docx` (Word) 파일로 한 번에 내보냅니다.
- **🌙 다크 모드**: 눈의 피로를 줄여주는 다크 모드와 다양한 한글 웹폰트를 지원합니다.

## 🚀 실행 방법 (How to Run)

이 프로젝트는 **단일 HTML 파일(Single File Application)**로 구성되어 있습니다.

1. 이 저장소의 `board.html` 파일을 다운로드합니다.
2. 다운로드한 파일을 크롬(Chrome), 엣지(Edge), 웨일(Whale) 등 최신 웹 브라우저로 엽니다.
3. 별도의 설치나 서버 실행 없이 즉시 사용 가능합니다!
   *(최초 실행 시 라이브러리 로드를 위해 인터넷 연결이 필요합니다.)*

<br/>

## 🧰 기술 스택 (Tech Stack)

별도의 빌드(Build) 과정 없이 브라우저에서 바로 실행되도록 구성되었습니다.

- **Core**: HTML5, React 18, ReactDOM
- **Styling**: Tailwind CSS (CDN)
- **Animation**: Framer Motion
- **Visualization**: Dagre (Graph Layout Algorithm)
- **Utils**: 
  - `CryptoJS` (AES 데이터 암호화)
  - `docx` & `FileSaver` & `JSZip` (워드 파일 생성 및 압축)
  - `Babel Standalone` (JSX 실시간 컴파일)

<br/>

## 💾 데이터 관리

- **Local Storage**: 작업 내용은 브라우저의 로컬 스토리지에 실시간으로 자동 저장됩니다.
- **File Export/Import**: 
  - `.json`: 일반 백업 파일
  - `.vel`: 비밀번호로 암호화된 보안 파일
