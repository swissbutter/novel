# 🧊 사각 스튜디오 (Sagak Studio)

> **웹소설 작가를 위한 올인원 비주얼 집필 도구** > 노드 구조의 마인드 맵 시스템과 집필 에디터의 완벽한 결합!

사각 스튜디오는 소설가, 시나리오 작가, 그리고 세계관 창작자를 위한 웹 기반 워크스페이스입니다. 인물 관계도, 사건의 흐름, 세계관 설정을 무한한 캔버스 위에서 시각적으로 정리하고, 이를 집필 에디터와 실시간으로 연동하여 창작 효율을 극대화합니다.

## ✨ 주요 기능 (Key Features)

### 1. ♾️ 무한 캔버스 & 보드 시스템 (Infinite Board)
* **자유로운 배치**: 줌(Zoom) & 팬(Pan) 기능을 통해 광활한 세계관을 한눈에 조망하거나 세부 설정에 집중할 수 있습니다.
* **13종의 특화 노드**: 인물, 사건, 장소, 아이템, 세력, 복선, 타임라인 등 소설 구성에 필수적인 속성값을 입력할 수 있습니다.
* **스마트한 연결 (Edges)**: 
    * 노드 간 관계를 드래그 앤 드롭으로 연결합니다.
    * **다양한 스타일**: 실선, 파선(Dashed), 점선(Dotted), 이중선(Double)을 지원하여 관계의 성격(혈연, 적대, 동맹 등)을 구분합니다.
* **자동 레이아웃**: 복잡하게 얽힌 노드들을 `Dagre` 알고리즘을 통해 논리적 순서로 즉시 자동 정렬합니다.

### 2. 📝 지능형 문서 에디터 (Integrated Editor)
* **실시간 설정 참조 (Mention)**: 집필 중 흐름을 깨지 않고 보드의 설정을 불러옵니다.
    * `@` 입력: 등장인물 설정 팝업 및 이름 자동 입력
    * `#` 입력: 주요 사건 및 타임라인 참조
    * `$` 입력: 세계관 설정 및 일반 메모 참조
    * `%` 입력: 할 일 목록
* **집중 모드 (Zen Mode)**: `Ctrl + J`를 눌러 활성화하며, 현재 집필 중인 라인이 화면 중앙에 고정되는 포커스 기능을 제공합니다.
* **문서 비교 (Split View)**: 사이드바에서 `Shift + 클릭`으로 두 문서를 동시에 열어 대조 집필이 가능합니다.

### 3. 🛡️ 데이터 보호 및 보안
* **원고 스냅샷**: 최대 20개의 히스토리를 자동 관리하여 실수로 삭제한 원고도 즉시 복구할 수 있습니다.
* **보안 저장 (.vel)**: 암호화 알고리즘(AES)을 적용하여 본인만 열어볼 수 있는 독자적인 파일 형식으로 내보낼 수 있습니다.
* **워드(Docx) 변환**: 작성한 원고와 보드 데이터를 MS Word 파일로 변환하여 투고 및 공유가 용이합니다.

## 📖 상세 사용 방법 (Usage Guide)

### 1. 작품 관리 (Library)
1. **새 작품**: '새 작품 시작하기' 버튼을 눌러 제목과 작가명을 입력합니다.
2. **커스텀**: 책 카드의 설정 아이콘에서 표지 색상을 변경하여 장르별로 구분하세요.
3. **불러오기**: 상단의 '파일 불러오기'를 통해 기존에 작업하던 `.json`이나 `.vel` 파일을 로드할 수 있습니다.

### 2. 보드 활용하기 (Planning)
1. **노드 생성**: 상단 툴바에서 원하는 타입(인물, 사건 등)을 선택해 캔버스에 배치합니다.
2. **관계 연결**: 노드 위에서 **마우스 오른쪽 버튼을 누른 채 다른 노드로 드래그**하면 관계선이 생깁니다.
3. **노드 편집**: 노드를 더블 클릭하여 이모지 변경, 상세 속성(나이, 직업 등)을 입력하세요. 🎲 아이콘을 누르면 랜덤 이름 생성기가 작동합니다.

### 3. 스마트 집필 (Writing)
1. **에디터 진입**: 사이드바에서 문서를 선택해 집필을 시작합니다.
2. **멘션 활용**: 등장인물이 나올 때 `@`를 입력해 보세요. 설정창을 따로 열지 않아도 캐릭터 정보를 확인할 수 있습니다.
3. **타이머 사용**: 우측 상단의 뽀모도로 타이머를 켜고 정해진 시간 동안 몰입하세요.

## ⌨️ 주요 단축키 (Hotkeys)

| 단축키 | 기능 |
| :--- | :--- |
| **`Ctrl + J`** | 집중 모드 (Zen Mode) 토글 |
| **`Ctrl + Z`** | 보드 작업 실행 취소 (Undo) |
| **`Shift + 클릭`** | (사이드바) 다중 문서 열기 / (보드) 다중 선택 |
| **`우클릭 드래그`** | 노드 간 관계선(Edge) 생성 |
| **`Delete`** | 선택한 노드 삭제 |

## 🛠 기술 스택 (Tech Stack)

* **Framework**: React 18, Framer Motion
* **Styling**: Tailwind CSS
* **Libraries**: Dagre (Graph Layout), CryptoJS (Security), Docx.js (Word Export)
* **Typography**: 네이버 마루부리, 리디바탕, 갈무리14
