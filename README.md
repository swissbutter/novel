# Sagak Studio
> **웹소설 작가를 위한 가장 진보된 도구.**
> 복잡한 세계관은 직관적으로 정리하고, 오직 글쓰기에만 몰입할 수 있는 환경을 제공합니다.

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

웹소설 집필 중 설정 붕괴나 집중력 저하로 고민하시나요?
**사각 스튜디오(Sagak Studio)**는 파편화된 아이디어를 시각적으로 구조화하는 **노드 시스템**과, 집필의 흐름을 끊지 않고 설정을 참조할 수 있는 **스마트 에디터**를 결합한 올인원 집필 도구입니다.

복잡한 설정은 '사각'에 맡기고, 작가님은 써 내려가는 즐거움에만 집중하세요.

---

## ✨ 핵심 기능 (Key Features)

### 1. 💡 상상을 시각화하는 [노드 시스템]
인물 관계도, 사건의 흐름을 더 이상 머릿속으로만 그리지 마세요.
- **직관적인 연결:** 인물(👤), 사건(🔥), 메모(💡) 노드를 생성하고 드래그 앤 드롭으로 관계를 설정합니다.
- **상세 설정:** 각 노드에 성별, 나이, 역할(주인공/적대자 등), 발생 시기/장소 등 상세 정보를 기록할 수 있습니다.
- **자동 정렬:** 복잡해진 노드들을 버튼 하나로 깔끔하게 정리합니다.

### 2. 📝 쓰면서 확인하는 [스마트 에디터 & 멘션]
설정집을 뒤적이느라 집필 흐름을 끊지 마세요.
- **즉각적인 멘션:** 에디터에서 `@`(인물) 또는 `#`(사건)을 입력하여 설정된 노드를 즉시 호출합니다.
- **호버 팝업:** 멘션된 키워드에 마우스를 올리면 상세 설정 내용이 팝업으로 표시되어 즉시 확인할 수 있습니다.

### 3. 🧘 오직 글과 나만 남는 시간, [집중 모드 (Zen Mode)]
- **단축키 실행:** `Ctrl + J` (Mac: `Cmd + J`)로 복잡한 인터페이스를 숨기고 오직 글쓰기 화면에만 집중할 수 있습니다.
- **다크 모드 & 폰트:** 눈이 편안한 다크 모드와 다양한 서체(마루부리, 갈무리 등)를 지원합니다.

### 4. 🔒 안전한 데이터 관리 및 내보내기
- **내 서재:** 여러 작품을 실제 책장처럼 직관적으로 관리합니다.
- **강력한 보안:** 비밀번호를 설정하여 암호화된 파일(`.vel`)로 안전하게 로컬에 저장합니다.
- **유연한 내보내기:** 보드 설정과 원고 내용을 각각 Word(`.docx`) 문서로 내보낼 수 있습니다.

## 🛠 기술 스택 (Tech Stack)
- **Framework:** React 18
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Graph Layout:** Dagre
- **Utilities:** Crypto-js (암호화), docx (문서 생성), FileSaver, JSZip
