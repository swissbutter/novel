# 🧊 Sagak Studio (사각 스튜디오)

> **웹소설 작가를 위한 올인원 비주얼 집필 도구**  
> 노드 구조의 마인드 맵 시스템 장착!

<img src="sagak_icon.png" width="250">

**Sagak Studio**는 소설가, 시나리오 작가, 그리고 세계관 창작자를 위한 웹 기반의 기획 도구입니다. 인물 관계도, 사건의 흐름, 세계관 설정을 무한한 캔버스 위에서 시각적으로 정리하고, 집중 모드를 지원하는 에디터에서 바로 집필할 수 있습니다.

## ✨ 주요 기능 (Key Features)

### 1. ♾️ 무한 캔버스 & 보드 시스템 (Infinite Board)
- **자유로운 배치**: 줌(Zoom) & 팬(Pan) 기능을 통해 광활한 작업 공간을 자유롭게 탐색할 수 있습니다.
- **다양한 노드 타입**: 
  - 👤 **인물**: 성별, 나이, 직업, 역할(주인공/적대자 등) 설정
  - 🔥 **사건**: 발생 시기, 장소 기록
  - 🎁 **아이템/장소/세력**: 세계관의 디테일한 요소 관리
  - 🎣 **복선/타임라인/갈등**: 스토리의 구조적 설계 지원
  - 📁 **그룹**: 노드들을 시각적으로 묶어 관리 (크기 조절 가능)
- **스마트한 연결 (Edges)**:
  - 노드 간 관계를 드래그 앤 드롭으로 연결
  - **다양한 선 스타일**: 실선, 파선(Dashed), 점선(Dotted), 이중선(Double) 지원
  - **동적 스타일링**: 선 굵기에 따라 점선/파선의 간격이 자동으로 조정되어 가독성 유지
  - 관계에 대한 메모(Label) 기능

### 2. 📝 강력한 문서 에디터 (Integrated Editor)
- **집중 모드 (Zen Mode)**: 주변 UI를 모두 숨기고 글쓰기에만 집중할 수 있는 환경 제공 (`Ctrl + J` 단축키)
- **문서 분할**: 문서를 Shift를 눌러 복수 선택하면 화면 분할로 비교 할 수 있습니다.
- **멘션 시스템**: `@인물`, `#사건`, `$메모` 입력 시 보드에 생성된 노드를 즉시 참조하거나 삽입 가능
- **뽀모도로 타이머**: 집필에 더욱 집중 할 수 있도록 타이머를 장착했습니다.
- **자동 저장**: 작성 중인 내용은 로컬 스토리지에 실시간으로 안전하게 저장됩니다. 하지만 주기적으로 로컬 저장하세요.

### 3. 🛠️ 유틸리티 & 편의 기능
- **🎲 이름 생성기**: 판타지(왕족/기사/마법사), 무협(세가/사파), 현대 한국 등 장르별/성별 랜덤 이름 생성 지원
- **자동 정렬 (Auto Layout)**: 복잡하게 얽힌 노드들을 보기 좋게 자동으로 정리 (Dagre 알고리즘 탑재)
- **다크 모드**: 눈의 피로를 줄여주는 다크 테마 완벽 지원
- **폰트 변경**: 마루부리, 리디바탕, 갈무리 등 가독성 좋은 한글 폰트 전환 기능

### 4. 🔒 데이터 관리 & 보안
- **강력한 내보내기/불러오기**:
  - `JSON`: 일반 백업 파일
  - `VEL`: **AES 암호화**가 적용된 보안 파일 (비밀번호 설정 가능)
  - `DOCX`: 기획 내용과 원고를 MS Word 문서로 변환하여 다운로드
- **단일 파일 구조**: 모든 코드가 하나의 HTML 파일에 포함되어 있어, 서버 없이도 실행 가능.

## 🚀 시작하기 (Getting Started)

1. `index.html` 파일을 다운로드합니다.
2. 파일을 크롬(Chrome), 엣지(Edge), 웨일(Whale) 등 최신 웹 브라우저로 엽니다.
3. **'새 작품 만들기'**를 클릭하여 프로젝트를 시작하세요.

## ⌨️ 단축키 (Shortcuts)

| 키 (Key) | 기능 (Action) |
| :--- | :--- |
| `Ctrl` + `Z` | 실행 취소 (Undo) |
| `Delete` / `Backspace` | 선택한 노드/연결 삭제 |
| `Ctrl` + `J` | 에디터 집중 모드 (Zen Mode) 토글 |
| `Tab` (분할 화면) | 분활 문서 위치 바꾸기 |
| `Shift` + `좌클릭` | 노드 복수 선택 or 문서 복수 선택|
| `Shift` + `좌클릭 드래그` | 노드 사각 영역 선택 |
| `Double Click` (노드) | 노드 상세 편집 |
| `Double Click` (선) | 관계선 상세 편집 |

## 💻 기술 스택 (Tech Stack)

이 프로젝트는 별도의 번들링 과정 없이 브라우저에서 즉시 실행되도록 설계되었습니다.

- **Frontend**: React 18, Tailwind CSS, Framer Motion
- **Layout Engine**: Dagre.js (Graph Visualization)
- **Security**: Crypto-JS (AES-256)
- **File Processing**: JSZip, FileSaver.js, docx.js

---

> **주의**: 데이터는 브라우저의 로컬 스토리지에 저장됩니다. 캐시 삭제 시 데이터가 유실될 수 있으므로, 주기적으로 [파일 저장] 기능을 통해 백업 파일을 생성하시기 바랍니다.
